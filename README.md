# Pequenos exercicios de java

Repositório dedicado ao armazenamento dos meus exercícios e projetos práticos durante o aprendizado de Java.

##  Projetos Inclusos

* **Calculadora IMC:** Cálculo de índice de massa corporal.
* **Sistema de Login:** Validação de credenciais de usuário.
* **Conversor de Moedas:** Prática com lógica matemática e tipos de dados.
* **Somador Numérico:** Exercícios básicos de entrada e saída.

##  Tecnologias
* Java JDK 25+
* IntelliJ IDEA

##  Como rodar
1. Clone o repositório: `git clone https://github.com/ViniciusCC-dev/Projetos-Java.git`
2. Abra a pasta no seu editor favorito (IntelliJ recomendado).
